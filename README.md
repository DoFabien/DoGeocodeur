# Dev
La nouvelle version utilise Angular 8, Angular Material et Leaflet 

```sh
git clone https://github.com/DoFabien/DoGeocodeur.git

cd DoGeocodeur
npm install

ng serve ## dev
ng build --prod ## version de prod

```
